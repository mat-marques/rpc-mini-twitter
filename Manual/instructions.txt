1) Baixar a biblioteca para usar o rpcgen
    - sudo apt-get install rpcbind
    - sudo apt install libc-dev-bin


2) Baixar os arquivos do projeto do mini-twitter

3) Baixar o Mysql
    - sudo apt-get install mysql-server mysql-client
    - sudo apt-get install libmysqlclient-dev libmysql++-dev libmysqlcppconn-dev
    
4) Configurar o banco de dados da aplicação seguindo as instruções no arquivo da pasta: 
SQL/Specifications/twitter.sql
    - Entrar no banco de dados: mysql -h localhost -u root -p
    - Copiar os comandos do arquivo para o terminal
    - Ao compilar qualquer extensão .c que tenha manipulação do mysql, então deve-se colocar
    a tag `mysql_config --cflags --libs`
